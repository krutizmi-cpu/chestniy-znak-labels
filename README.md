# Генератор этикеток Честный Знак (DataMatrix)

Онлайн-сервис на Streamlit для создания и печати этикеток с кодами маркировки Честного знака.

## Возможности
- **Массовая загрузка**: Загрузка кодов через Excel (шаблон прилагается).
- **Одиночный ввод**: Быстрое создание одной этикетки вручную.
- **Разные форматы**: Поддержка 58x40, 58x60, 40x30, 100x50, A6, A4.
- **Умная валидация**: Проверка структуры КИЗ и предупреждение об отсутствии криптохвоста.
- **Гибкий экспорт**: Генерация PDF (поштучно или сеткой на A4) и Excel-отчетов.

## Установка и запуск

1. Клонируйте репозиторий:
```bash
git clone https://github.com/krutizmi-cpu/chestniy-znak-labels.git
cd chestniy-znak-labels
```

2. Установите зависимости:
```bash
pip install -r requirements.txt
```

3. Запустите приложение:
```bash
streamlit run app.py
```

## Требования
- Python 3.9+
- Ghostscript (обязательно для работы библиотеки `treepoem` и генерации DataMatrix)
